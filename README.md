## requirements

- cargo
  - install with `curl https://sh.rustup.rs -sSf | sh` or use install script of skyfetch

## installation 

```
git clone https://github.com/justleoo/skyfetch
cd skyfetch
cargo install --path .
```

and use `skyfetch` command in your terminal!
